# ChillOS v0.8
---

ChillOS — это веб-симуляция операционной системы, работающая прямо в браузере.
Проект имитирует процесс запуска BIOS, загрузку системы, рабочий стол в стиле Windows 11, панель задач, анимации, звуки включения и выключения, а также обработку системных ошибок (чёрный экран смерти).
Система управляется через JS файл, что позволяет изменять поведение загрузки, интерфейс, обои, звуки и другие параметры. ChillOS — это экспериментальный UI-проект, созданный для исследования логики состояний, анимаций, системных сценариев и архитектуры “ОС внутри браузера”.
Проект написан на чистом HTML, CSS и Vanilla JavaScript без использования фреймворков.
