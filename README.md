**ChillOS** - это операционная система внутри браузера с открытым исходным кодом, которая была вдохновлена проектом `windows93.net`, однако с современным дизайном. Пока что проект во многом малофункционален, однако планируется расширение функций, и в конце концов выпуск полноценного релиза 1.0, который перейдёт в состояние бета тестирования. Надеюсь, вам проект понравится :)
